# Jonril Documentation

Jonril is a workspace-first note system that combines structured journals, linked notes, and task workflows in one place.

## What Jonril is for

- Capture ideas quickly in notes and journal entries.
- Turn note content into actionable tasks.
- Navigate knowledge through links, headings, and backlinks.
- Keep work organized per workspace.

## Core concepts

### Notes

Regular notes are hierarchical and can contain rich content such as headings, lists, links, mentions, hashtags, and tasks.

### Journals

Journal notes are date-based and available by:

- Daily
- Weekly
- Monthly
- Yearly

### Tasks

Tasks are embedded in notes and can be filtered, grouped, migrated, and tracked across the app.

## Getting started

1. Open **Notes** to create or browse regular notes.
2. Open **Journal** to work in date-based notes.
3. Use **Tasks** to review and manage work across notes.
4. Use `Cmd/Ctrl + K` to open the command palette.
